Chiptuning SEO plugin extension for WordPress to enhance search engine optimization.

== Description ==

This plugin extends the functionality of the Chiptuning Reseller plugin by adding SEO features.

With this extension, you can set up:

*   default SEO settings: title, description, H1, bottom text
*   specific SEO settings for every brand/serie/model/engine: title, description, H1, bottom text
